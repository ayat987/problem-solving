Welcome to my **Problem Solving** repository!  

This repository contains my solutions for competitive programming problems from platforms like **Codeforces**, **LeetCode**, and **AtCoder**.  
The goal is to track my progress, improve my problem-solving skills, and organize solutions by topics and difficulty.

## 📂 Structure

- `Codeforces/` - Problems solved from Codeforces
- `LeetCode/` - Problems solved from LeetCode
- `AtCoder/` - Problems solved from AtCoder
- `Topics/` - Problems organized by topics like DP, Greedy, Graphs, etc.
